# Local Infrastructure Documentation

This documentation provides guidance for deploying the Wanderlust DevSecOps project on local infrastructure.

## Tested Environment

### VM-1

* 8 GB RAM
* 80 GB Storage

Services:

* Jenkins
* SonarQube
* OWASP Dependency Check
* Trivy
* Docker
* Kubernetes Control Plane

Services:

* Kubernetes Worker Node
* ArgoCD
* Prometheus
* Grafana
